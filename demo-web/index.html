ICAP demo • REQMOD + RESPMOD

Security Modules, Visible.

This demo web server is designed to trigger ICAP request and response inspection. Use the sample payloads to validate header/body rewrites, URL checks, and antivirus scanning.

REQMOD

URL reputation check

curl -v http://demo.local/blocked?url=http://example.com/bad
Expected: ICAP url_check service decides block/pass based on DNSBL.
RESPMOD

Response content rewrite

curl -v http://demo.local/echo?body=ORIGINAL
Expected: body token rewritten, header X-ICAP-Rewritten: 1.
REQMOD / RESPMOD

Antivirus scan

curl -v http://demo.local/upload --data-binary @eicar.txt
Expected: ICAP virus_scan checks payload against clamd.
OBSERVABILITY

Syslog audit trail

docker exec cicap_dev tail -n 20 /var/log/syslog
Expected: syslog entries per ICAP request.
